## Filters

### `pronamic_money_default_format`

*Filters the default money format.*

Default format: `%%1$s%%2$s %%3$s`

- 1: Currency symbol
- 2: Amount value
- 3: Currency code

Note: use non-breaking space ` ` in money formatting.

**Arguments**

Argument | Type | Description
-------- | ---- | -----------
`$format` | `string` | Format.
